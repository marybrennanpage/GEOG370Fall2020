In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

Using QGIS, I was able to display the map using different projections. First, I downloaded the data from Sakai to my desktop. After I saved and unzipped the files, I opened QGIS to begin working. Once in QGIS, I uploaded the ne_10m_admin_0_countries.shp (shapefile). Then I was able to click on the EPSG on the bottom right of the screen to adjust/change the EPSG for the desired coordinate reference system. For example, when I wanted to create the Pseudo-Mercator Projection, I clicked on the EPSG and filtered for "3857", and selected the Pseudo-Mercator coordinate reference system. After selecting the desired coordinate reference system, I used the Indicatrix Mapper plug-in, running the defaults, which added indicatrix circles over the projections. The indicatrix circles are useful to see the distortions that each projection cause.
